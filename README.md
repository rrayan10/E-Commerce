Modern E-Commerce Store
- A sleek, fast, and responsive E-commerce frontend application. Built using React, Vite, and styled with Shadcn/UI and Tailwind CSS.
- Live demo: https://e-commerce-phi-five-32.vercel.app/

Features
- Modern UI/UX: Clean and intuitive design using Shadcn/UI components.
- Fully Responsive: Optimized for mobile, tablet, and desktop screens.
- Fast Performance: Powered by Vite for an ultra-fast development experience.
- Dynamic Routing: Seamless navigation throughout the store.

Tech Stack
- React & NextJS
- React Hook
- Vite
- TypeScript
- Styling & UI: Tailwind CSS, Shadcn/UI & Lucide React

Getting Started
Follow these steps to run the project locally on your machine:
- Clone the repository
- Navigate to the project folder
- Install dependencies
- Start the development server

Scripts
- npm run dev
- npm run build
- npm run preview
